# Tic Tac Toe GUI

A desktop Tic Tac Toe game built with Python and PyQt6. Two players take turns playing X and O on a 3x3 grid, with the game automatically detecting wins, losses, and draws.

## Features

- Interactive GUI built with PyQt6
- Two-player Tic Tac Toe on a 3x3 grid
- Automatic win and draw detection for rows, columns, and diagonals
- Color-coded moves (X in green, O in blue)
- Tracks and displays score for X wins, O wins, and draws
- Play Again button to start a new round
- Clear Stats button to reset the scoreboard

## Technologies Used

- Python
- PyQt6

## How to Run

1. Clone or download the repository
2. Install dependencies:
```
   pip install -r requirements.txt
```
3. Run the game:
```
   python mycode.py
```
